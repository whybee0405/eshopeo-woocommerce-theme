# K-BAP Logo Guidelines

## Core Requirement

Do not redesign the K-BAP logo. The logo represents a slice of gimbap.

- Dark outer section: seaweed.
- White inner section: rice.
- Coloured elements: fillings.

The recognisable silhouette must remain intact. Improvements may only address alignment, spacing, geometry, balance, scalability, and vector quality.

## Current Issue

The current theme placeholder uses a red circular "KB" mark. That is not the approved K-BAP logo concept and should be replaced during the logo refinement increment.

## Logo Principles

- Preserve the gimbap-slice idea.
- Keep the silhouette simple enough for favicon use.
- Avoid bowl shapes.
- Avoid chopsticks, flags, mascots, anime marks, or decorative Korean motifs.
- Use geometry that scales cleanly from favicon to packaging.
- Keep typography secondary to the gimbap mark.

## Required Variants

### Primary

Gimbap slice mark with K-BAP wordmark. Use on website header, homepage, catering material, and packaging front panels.

### Horizontal

Mark left, wordmark right. Use for website header, invoices, email, and narrow layouts.

### Vertical

Mark above wordmark. Use for packaging, social avatars, and centered brand applications.

### Monochrome

Single-colour mark and wordmark. Use for stamps, receipts, thermal labels, embossing, and low-cost print.

### Dark

Rice or warm light mark treatment on Seaweed backgrounds.

### Light

Seaweed mark treatment on Rice backgrounds.

### Favicon

Mark only. Minimum 32px. Fillings simplified if required for clarity.

### App Icon

Mark centered on Rice background. Keep enough padding for rounded OS masks.

### Packaging Version

Mark and wordmark with stronger contrast and clear minimum size rules.

### Social Version

Mark dominant, wordmark optional. Must remain recognisable in a circular crop.

## SVG Specification

The refined SVG should:

- Use real vector shapes, not embedded raster images.
- Avoid text converted inconsistently across exports.
- Use `viewBox` values that create predictable scaling.
- Avoid filters, gradients, masks, and unnecessary groups.
- Keep paths named or grouped logically.
- Use current brand tokens where possible.

## Clear Space

Minimum clear space around the logo equals the width of the rice ring. On packaging, increase clear space when the logo sits near product names or nutrition panels.

## Minimum Sizes

- Full horizontal logo: 120px wide on screen.
- Primary stacked logo: 96px wide on screen.
- Mark only: 24px minimum, 32px preferred.
- Packaging front: test at actual shelf distance before final print.

## Incorrect Use

Do not:

- Change the mark into a bowl.
- Place the mark inside a red circle.
- Add chopsticks or steam.
- Add gradients.
- Add shadows.
- Rotate the mark casually.
- Stretch the wordmark.
- Use neon, pastel, or unapproved colours.
